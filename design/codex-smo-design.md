# SMO Module 前端页面设计 (对象线5号: SMO / ProjectSMO)

> SMO运营域完整设计，含三入口页面、公共组件规范

---

## 一、三界面对照

| 层级 | 菜单位置 | JS文件 | 视角 | 操作对象 |
|------|---------|--------|------|---------|
| L1-SMO主档 | 综合管理>合作主体>SMO管理 | smo-master.js | 全平台 | SMO主数据+合作历史 |
| L2-项目SMO | 项目中心>某项目>SMO管理 | project-smo.js | 单项目 | ProjectSMO+CRC分配+费用/付款 |
| L3-中心SMO | 中心驾驶舱(嵌入) | project-site-smo.js | 单中心 | 中心关联SMO信息(只读) |

- CLOSED → 不可变更

### 3.8 费用管理入口

- **[→费用管理]：** 跳转至费用管理页面，筛选条件预填 `payee_type='SMO'&payee_id=:smoId&project_id=:projectId`
- **[→付款管理]：** 跳转至付款管理页面

### 3.9 绩效评分

**按项目评分，数据来自项目SMO记录：**
- 入组贡献率：该项目SMO覆盖中心的入组数 / 项目总入组数
- 数据质量：由QA/PM评价
- 响应速度：由PM评价
- 综合评分：加权平均

## 五、公共组件规范

### 5.1 SMO选择器 (SmoPicker)

**使用场景：** 新增项目SMO、费用关联SMO

**交互：**
- 输入框，搜索SMO名称/简称
- 下拉显示：short_name / smo_name / rating / crc_count
- 支持单选

**props：**
```javascript
{
  placeholder: '搜索SMO名称/简称',
  filters: { status: 'ACTIVE', rating: null },
  onChange: (selected) => {}
}
```

### 5.2 中心选择器 (SitePicker)

**使用场景：** 新增项目SMO-选择覆盖中心、CRC分配中心

**交互：**
- 多选列表，展示中心名称+状态
- 已关联中心置灰不可选

**props：**
```javascript
{
  projectId: '',
  excludeSiteIds: [],  // 已关联的中心ID
  mode: 'single' | 'multi',
  onChange: (selected) => {}
}
```

### 5.3 星级评分组件 (StarRating)

**使用场景：** SMO评级选择、绩效评分展示

**交互：**
- 点击/hover切换星级
- 支持半星(0.5步进)
- 显示当前分数

**props：**
```javascript
{
  value: 4.5,
  max: 5,
  readonly: false,
  onChange: (rating) => {}
}
```

### 5.4 资质文件列表 (QualificationList)

**使用场景：** SMO资质管理、详情展示

**交互：**
- 表格展示：文件类型/文件名/上传日期/有效期/状态
- 状态Tag：VALID=绿 / EXPIRING=黄(30天内) / EXPIRED=红
- 支持上传/下载/删除

**props：**
```javascript
{
  entityType: 'SMO' | 'MEMBER' | 'SITE',
  entityId: '',
  docs: [],
  editable: true,
  onUpload: (file) => {},
  onDelete: (docId) => {}
}
```

### 5.5 合同关联组件 (ContractLinker)

**使用场景：** 项目SMO关联合同

**交互：**
- Tab切换：[上传新合同] [关联已有合同]
- 上传：表单+文件
- 关联：搜索已有合同，选择后自动填充

**props：**
```javascript
{
  contractType: 'SMO',
  projectId: '',
  linkedContractId: null,
  onLink: (contractId) => {},
  onUpload: (formData) => {}
}
```
