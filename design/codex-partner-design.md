# 其他合作方模块前端页面设计 (对象线7号: Partner / ProjectPartner)

> 其他合作方是CRO临床研究SaaS平台供应链模块下的兜底分类
> 覆盖：检测公司、IRC(独立评审委员会)、中心检测实验室、数据管理、统计分析等特殊合作主体
> 核心逻辑复用SMO模式，关键差异：partner_type字段区分类型

---

## 一、三界面对照

| 层级 | 菜单位置 | JS文件 | 视角 | 操作对象 |
|------|---------|--------|------|---------|
| L1-合作方主档 | 综合管理>供应链>其他合作方管理 | partner-master.js | 全平台 | 合作方主数据+合作历史 |
| L2-项目合作方 | 项目中心>某项目>其他合作方 | project-partner.js | 单项目 | ProjectPartner+合作项目 |
| L3-中心合作方 | 中心驾驶舱(嵌入) | project-site-partner.js | 单中心 | 中心关联合作方信息(只读) |

### 2.8 穿透导航路径

**L1→项目穿透：**
- 合作项目Tab中，点击项目名称行 → 打开项目详情页（路由：项目中心 > {项目编号}）
- 点击项目行中的"合作中心"数字 → 展开该合作方在此项目下的中心列表，可进一步点击进入中心

**L1→中心穿透：**
- 覆盖中心数字可点击 → 弹出该合作方覆盖的中心列表
- 点击中心名称 → 进入中心驾驶舱页面（路由：项目中心 > {项目编号} > 中心管理驾驶舱 > 其他合作方）
- 面包屑自动回溯：从L1穿透进入时面包屑显示 `综合管理 > 合作方 > 金域检测 > 北京协和医院`

**穿透路径总结：**
```
L1主档 → 点击项目名称 → L2项目合作方详情（带面包屑回溯）
L1主档 → 点击中心名称 → L3中心合作方详情（带面包屑回溯）
L2项目 → 点击中心名称 → L3中心合作方详情
L2项目 → 点击合同状态 → 合同详情（合同对象线）
L3中心 → 点击合同状态 → 合同详情（合同对象线）
```

### 2.9 合同入口（预留对接接口）

> **重要**：合同详情在合同对象线中实现，本模块只保留入口，确保可对接

**合同入口位置与行为：**

| 入口位置 | 展示内容 | 点击行为 | 对接参数 |
|---------|---------|---------|---------|
| L1详情-合同Tab | 关联合同列表(合同编号、金额、状态、有效期) | → 跳转合同详情页 | partner_id + contract_id |
| L2项目合作方-合同信息 | 当前项目关联合同(编号、金额、履行进度条) | → 跳转合同详情页 | project_partner_id + contract_id |
| L2项目合作方-详情弹窗 | 合同金额/已执行/状态 | → 跳转合同详情页 | contract_id |
| L3中心合作方-合同状态 | 合同状态Tag(有效/即将到期/已到期) | → 跳转合同详情页 | contract_id |

**合同状态Tag规范（全局统一）：**
- ● 有效 = 绿色(#059669)
- ● 即将到期 = 橙色(#D97706)，30天内到期
- ● 已到期 = 红色(#DC2626)
- ● 待签署 = 灰色(#6B7280)

**合同详情页路由（合同对象线实现后对接）：**
```
合同管理 > {contract_id}
或项目中心 > {项目编号} > 合同 > {contract_id}
```

**对接规范：**
- 所有合同入口统一使用 `data-contract-id` 属性，合同对象线实现后绑定全局点击事件
- 合同列表数据通过 `/api/contract/?partner_id=xxx` 或 `/api/contract/?project_id=xxx` 获取
- 不在本模块实现合同CRUD

## 五、公共组件规范

### 5.1 合作方选择器 (PartnerPicker)

**使用场景：** 新增项目合作方、费用关联合作方

**交互：**
- 输入框，搜索合作方名称/简称
- 按类型分组展示：LAB / IRC / DATA_MANAGER / STAT_ANALYST / OTHER
- 下拉显示：short_name / partner_name / partner_type / rating
- 支持单选

**props：**
```javascript
{
  placeholder: '搜索合作方名称/简称',
  filters: { partner_type: null, status: 'ACTIVE', rating: null },
  onChange: (selected) => {}
}
```

### 5.2 合作方类型标签 (PartnerTypeTag)

**使用场景：** 列表类型列、详情页类型显示

**类型映射：**
| partner_type | 显示名称 | 颜色 |
|-------------|---------|------|
| LAB | 检测公司 | #0891B2 (蓝色) |
| IRC | IRC评审 | #8B5CF6 (紫色) |
| DATA_MANAGER | 数据管理 | #059669 (绿色) |
| STAT_ANALYST | 统计分析 | #F59E0B (橙色) |
| OTHER | 其他 | #6B7280 (灰色) |

### 5.3 中心选择器 (SitePicker)

**使用场景：** 新增项目合作方-选择覆盖中心

**交互：**
- 多选列表，展示中心名称+状态
- 已关联中心置灰不可选

### 5.4 星级评分组件 (StarRating)

**使用场景：** 合作方评级选择、绩效评分展示

**交互：**
- 点击/hover切换星级
- 支持半星(0.5步进)
- 显示当前分数

### 5.5 资质文件列表 (QualificationList)

**使用场景：** 合作方资质管理、详情展示

**交互：**
- 表格展示：文件类型/文件名/上传日期/有效期/状态
- 状态Tag：VALID=绿 / EXPIRING=黄(30天内) / EXPIRED=红
- 支持上传/下载/删除

### 5.6 合同入口组件 (ContractEntry)

**使用场景：** 合作方详情-合同Tab、项目合作方-合同信息

**交互：**
- 显示合同关联入口
- 列表展示已关联合同(只读)
- **[关联合同]：** 跳转至合同模块
- **[上传合同]：** 跳转至合同模块新增
