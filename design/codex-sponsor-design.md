# Sponsor Module 前端页面设计 (对象线4号: Sponsor / ProjectSponsor)

> 申办方运营域完整设计，含两入口页面、公共组件规范
> **注意：合同/收款/发票管理已移至独立合同对象线，申办方模块仅保留跳转入口**

---

## 一、三界面对照

| 层级 | 菜单位置 | JS文件 | 视角 | 操作对象 |
|------|---------|--------|------|---------|
| L1-申办方主档 | 综合管理>合作主体>申办方管理 | sponsor-master.js | 全平台 | Sponsor主数据+合作项目 |
| L2-项目申办方 | 项目中心>某项目>申办方信息 | project-sponsor.js | 单项目 | ProjectSponsor+合同入口 |

## 四、公共组件规范

### 4.1 申办方选择器 (SponsorPicker)

**使用场景：** 新增项目申办方关联

**交互：**
- 输入框，搜索申办方名称/简称
- 下拉显示：short_name / sponsor_name / sponsor_type / vip_level / rating
- 支持单选

**props：**
```javascript
{
  placeholder: '搜索申办方名称/简称',
  filters: { sponsor_status: 'ACTIVE', sponsor_type: null },
  onChange: (selected) => {}
}
```

### 4.2 星级评分组件 (StarRating)

**使用场景：** 申办方评级选择、绩效评分展示

**交互：**
- 点击/hover切换星级
- 支持半星(0.5步进)
- 显示当前分数

**props：**
```javascript
{
  value: 4.5,
  max: 5,
  readonly: false,
  onChange: (rating) => {}
}
```

### 4.3 资质文件列表 (QualificationList)

**使用场景：** 申办方资质管理、详情展示

**交互：**
- 表格展示：文件类型/文件名/上传日期/有效期/状态
- 状态Tag：VALID=绿 / EXPIRING=黄(30天内) / EXPIRED=红
- 支持上传/下载/删除

**props：**
```javascript
{
  entityType: 'SPONSOR',
  entityId: '',
  docs: [],
  editable: true,
  onUpload: (file) => {},
  onDelete: (docId) => {}
}
```

### 4.4 合同入口卡片 (ContractEntryCard)

**使用场景：** 项目申办方中展示合同概览和跳转入口

**交互：**
- 显示合同份数、合同总额、已收款、待收款
- 提供"[查看合同 →]"按钮跳转至合同管理模块

**props：**
```javascript
{
  projectId: '',
  sponsorId: '',
  contractCount: 0,
  totalAmount: 0,
  collectionAmount: 0,
  onNavigate: () => {}
}
```

### 4.5 开票信息卡片 (InvoiceInfoCard)

**使用场景：** 申办方详情-开票信息展示

**交互：**
- 展示发票抬头、税号、银行账户等开票信息
- 支持编辑

**props：**
```javascript
{
  sponsorId: '',
  invoiceInfo: {},
  editable: true,
  onEdit: () => {}
}
```
