# HR 薪资管理模块设计文档 (salary-design.md)
> 数据库: salary.db (SQLite, better-sqlite3, WAL模式)
> 路由前缀: /api/salary/
> 设计依据: OA薪资历史数据(10553行) + 现有member_salary极简表(12列)扩充

---

## 一、模块定位

薪资管理模块是HR域的独立子模块，负责：
- 员工薪资结构定义与维护
- 月度发薪批次全流程（生成→审批→发放）
- 社保公积金配置与计算
- 个税累计预扣法计算
- 薪资历史与变更追踪
- 薪资报表（月度汇总/部门统计/年度累计）

**跨域依赖：**
- 人员数据(member)：通过HTTP API获取 /api/hr/members，不共享DB
- 审批流程(admin)：薪资批次审批走 /api/admin/approval，不共享DB

## 二、核心概念

### 2.1 薪资结构

一个员工的完整薪资由以下部分组成：

```
应发合计 = 基本工资 + 岗位工资 + 绩效工资 + 职务工资 + 各类补贴
扣款合计 = 考勤扣款 + 其他扣款
社保公积金(个人) = 养老(个人) + 医疗(个人) + 失业(个人) + 公积金(个人)
个税 = 累计预扣法计算
──────────────────────────────────────────
实发工资 = 应发合计 - 扣款合计 - 社保公积金(个人) - 个税
```

**公司成本：**
```
公司成本 = 应发合计 + 社保公积金(公司) + 残保金
社保公积金(公司) = 养老(公司) + 医疗(公司) + 失业(公司) + 工伤(公司) + 生育(公司) + 公积金(公司)
```

### 2.3 发薪批次

每月的发薪流程：

```
创建批次(年月) → 生成薪资明细(基于薪资结构+考勤数据) → 审批 → 发放
       ↑                                                           ↓
  DRAFT → GENERATED → PENDING_APPROVAL → APPROVED → PAID
```

- **DRAFT**: 批次创建，未生成明细
- **GENERATED**: 系统根据薪资结构+考勤自动生成明细，可人工调整
- **PENDING_APPROVAL**: 提交审批
- **APPROVED**: 审批通过
- **PAID**: 已发放
- **CANCELLED**: 作废

**批次操作流转：**
1. **生成明细**：DRAFT/GENERATED → GENERATED
   - 调用HR API获取在职人员列表
   - 根据salary_structure(当前生效) + salary_social_insurance + salary_tax_config 自动计算
   - 生成salary_record明细
   - 已有明细的员工不重复生成（支持追加新入职人员）
2. **提交审批**：GENERATED → PENDING_APPROVAL
   - 调用审批API创建审批单
3. **审批通过**：PENDING_APPROVAL → APPROVED（审批回调）
4. **确认发放**：APPROVED → PAID
   - 标记batch_status=PAID，记录paid_at
5. **作废**：任意状态 → CANCELLED（仅DRAFT/GENERATED可作废）

## 四、计算规则

### 4.1 月度薪资计算流程

```
1. 获取员工当前薪资结构(salary_structure, is_current=1)
2. 获取员工社保公积金配置(salary_social_insurance)
3. 计算应发合计 = base_salary + position_allowance + rank_salary + duty_salary + 各项补贴
4. 计算社保公积金
   - 确定缴纳基数: MIN(MAX(应发合计, base_min), base_max)
   - 公司部分 = 缴纳基数 × 各项公司比例
   - 个人部分 = 缴纳基数 × 各项个人比例
5. 计算税前金额 = 应发合计 - 扣款合计 - 社保公积金(个人)
6. 计算个税(累计预扣法)
   - 累计收入 = 本月税前 + 年初至上月累计税前
   - 累计免税额 = 5000 × 累计月数
   - 累计专项扣除 = 社保公积金(个人) × 累计月数
   - 累计应纳税所得额 = 累计收入 - 累计免税额 - 累计专项扣除
   - 查税率表 → 累计应纳税额 = 累计应纳税所得额 × 税率 - 速算扣除数
   - 本月个税 = 累计应纳税额 - 已累计缴税
7. 计算实发 = 税前金额 - 本月个税
```

### 4.2 个税累计预扣法要点

- 按年累计，每月预扣
- 起征点: ¥5,000/月（¥60,000/年）
- 专项附加扣除: 子女教育/继续教育/大病医疗/住房贷款/住房租金/赡养老人（预留字段，当前版本按0计算）
- 1月重置累计值
- 个税不能为负数，最低为0

### 5.1 OA老数据迁移

OA的 oa_staff_paymentinfo (10553行) 迁移策略：
